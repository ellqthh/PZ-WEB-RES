Что такое Web-дизайн?

Web-дизайн — это графический дизайн, перенесённый в интерактивную среду. Главная цель — помочь пользователю быстро и интуитивно сориентироваться на сайте.

Цель интерактивного Web-дизайна: сделать сайт полезным, визуально приятным, простым и понятным в использовании.

Основы композиции для Web

Равновесие

Состояние формы, при котором все элементы сбалансированы между собой

Симметрия

Строгость, спокойствие, классицизм, сила. Соответствует "важным" формам

Асимметрия

Движение, динамизм, жизнь, свобода, выразительность

Важные параметры оценки web-проекта:

  1. Скорость загрузки
  2. Уровень дизайна
  3. Эргономика
  4. Посещаемость
  5. Уровень контента

UX/UI дизайн и дизайн-мышление

UX-дизайн

User Experience — "пользовательский опыт". Отвечает за то, как интерфейс работает.

  • Исследование пользователей
  • Прототипирование
  • Тестирование удобства

UI-дизайн

User Interface — "пользовательский интерфейс". Отвечает за то, как интерфейс выглядит.

  • Визуальный дизайн
  • Цветовая схема
  • Типографика

Этапы дизайн-мышления:

1

Эмпатия

Понять пользователя, его потребности и проблемы

2

Фокусировка

Определить ключевую проблему для решения

3

Идеи

Генерация множества решений

4

Прототип

Создание модели для тестирования

5

Тест

Проверка решения с пользователями

Цвета в Web-дизайне

Красный

Энергия, страсть, опасность

Бирюзовый

Спокойствие, чистота, технологии

Жёлтый

Радость, оптимизм, внимание

Зелёный

Природа, рост, гармония

Оранжевый

Энергия, тепло, энтузиазм

Голубой

Мир, доверие, стабильность

Советы по работе с цветом:

  • Используйте 3-4 основных цвета на сайте
  • Текст должен контрастировать с фоном
  • Учитывайте цветовые ассоциации
  • Проверяйте сочетаемость цветов

Этапы работы над web-проектом

1 Планирование

Разработка структуры проекта, системы навигации

2 Дизайн

Создание дизайн-макета, прототипа интерфейса

3 Вёрстка

Превращение макета в HTML/CSS код

4 Программирование

Добавление функциональности (JavaScript, PHP и др.)

5 Тестирование

Проверка работы на разных устройствах и браузерах

6 Запуск

Размещение сайта на хостинге, наполнение контентом

4 правила профессионального дизайна

Детали

Добавляйте мелкие детали в элементы дизайна, визуально разбивайте крупные объекты

Внимание к мелочам

Продумывайте каждый элемент до конца, следите за сочетаемостью

Гармония цветов

Сочетание цветов должно быть комфортным для чтения и восприятия

Гармония шрифтов

Используйте 1-2 шрифта на сайте, избегайте декоративных гарнитур
